# wiki_notes

一个基于 Obsidian 的个人知识库仓库，用于整理原始摘录、结构化概念卡片、关系图谱和写作提示词。

## 目录结构

- `wiki/`：整理后的知识库内容、概念卡片、关系数据与可视化文件
- `raw/`：原始素材、摘录文本、抽取结果
- `prompts/`：知识整理流程中使用的提示词模板
- `.obsidian/`：Obsidian 配置文件

## 使用方式

1. 用 Obsidian 打开本仓库目录。
2. 在 `raw/` 中存放原始素材或摘录。
3. 根据需要使用 `prompts/` 中的提示词进行抽取、写作、索引更新和审查。
4. 将整理后的内容沉淀到 `wiki/` 中。
